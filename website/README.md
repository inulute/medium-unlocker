# Medium Unlocker Website

Modern, sleek React website for Medium Unlocker with the same ultra-modern dark UI as the Android app.

## Features

- Modern dark theme (#050505 background)
- Hero typography with tight letter spacing
- Download Android app button
- Unlock articles directly from the website
- GitHub star integration
- Support button
- Responsive design

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Tech Stack

- React 18
- Modern CSS with dark theme
- Inter font family
- Responsive design
